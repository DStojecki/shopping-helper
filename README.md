# shopping-helper

## Project setup
```
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python .\manage.py migrate

cd .\frontend\
npm install
```

### Compiles and hot-reloads for development
```
venv\Scripts\activate 
python .\manage.py runserver 8080

npm run serve
```

### Compiles and minifies for production
```
npm run build
```
